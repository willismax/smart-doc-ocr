# 智慧文件辨識系統 — 使用手冊（文書人員版）

## 開始使用

1. 雙擊桌面上的 **start.bat**（黑色視窗會打開，請不要關閉它）
2. 打開瀏覽器（Edge 或 Chrome），輸入網址：**localhost:8501**
3. 看到「智慧文件辨識系統」畫面就可以開始用了

> 🔒 所有文件都在這台電腦上處理，不會上傳到網路，請安心使用。

---

## 功能一：單檔辨識（把文件變成文字）

1. 左邊選單點「📄 單檔辨識」
2. 把文件**拖進**上傳框（或點擊選檔）
3. 按「🚀 開始辨識」
4. 等待完成（掃描文件比較慢，約 3~10 秒一頁）
5. 按「⬇️ 下載 Markdown」存檔

**支援的文件**：PDF、Word、Excel、PowerPoint、照片/掃描圖（JPG、PNG、TIFF）、電子郵件

---

## 功能二：文件比對（找出兩份文件的差異）

1. 左邊選單點「🔍 文件比對」
2. 左邊放**文件 A**（舊版/基準版），右邊放**文件 B**（新版）
3. 選比對方式：

   | 比對方式 | 什麼時候用 |
   |---|---|
   | 逐行文字差異 | 想知道**哪裡被改了**（合約改版、公文修訂） |
   | 語意相似度 | 想知道兩份文件**意思像不像**（換句話說也算像） |
   | 欄位比對 | 只想核對**特定欄位**（姓名、金額、日期…） |

4. 按「🔍 開始比對」

**看懂差異畫面**：紅色（開頭是 `-`）是文件 A 有、B 沒有的；綠色（開頭是 `+`）是 B 新增的。

---

## 功能三：批量處理（一次處理很多份）

1. 左邊選單點「📚 批量處理」
2. 一次選取多份文件（可以按住 Ctrl 多選）
3. 按「🚀 批量處理」
4. 完成後可下載：**報告**（哪些成功哪些失敗）和**全部辨識結果（ZIP）**

> 💡 某一份文件壞掉不會影響其他文件，失敗的會列在報告裡並說明原因。

---

## 關於「自動遮蔽個資」

- 左邊選單的「自動遮蔽個資」**預設是開的**：身分證字號、手機、地址等
  會顯示成《身分證字號》這樣的標籤，不會露出真實內容。
- 如果工作需要看原文，把開關關掉即可——但**關閉動作會被記錄**，
  請依單位規定使用。

---

## 常見問題

| 狀況 | 怎麼辦 |
|---|---|
| 畫面顯示「檔案有密碼保護」 | 先用原程式解除密碼再上傳 |
| 顯示「不支援的檔案格式」 | 確認是支援的格式；把檔案另存成 PDF 通常可解決 |
| 掃描文件辨識結果亂七八糟 | 重新掃描：放正、300 DPI、避免反光陰影 |
| 顯示「OCR 引擎未就緒」 | 數位 PDF 和 Word/Excel 照常可用；掃描檔請通知資訊人員 |
| 網頁打不開 | 確認黑色視窗還開著；重新雙擊 start.bat |
| 處理很慢 | 掃描檔每頁約 3~10 秒是正常的；批量時可先去做別的事 |

還是解決不了？請把畫面截圖傳給資訊人員，並告知檔案名稱與時間。
